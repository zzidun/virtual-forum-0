# 规范

## 命名

总体使用`snake_case`方式命名,以下例外:

1. 因为go的强制而使用`CamelCase`命名public标识符

2. 库本身的命名

| 类型 | 规则 |
| --- | --- |
| 文件 | file |
| 包 | package |
| 类 | TheClass |
| 私有成员 | the_field |
| 公共类员 | The_Field |
| 私有函数 | type_operator |
| 公共函数 | Type_Operator |
| 变量 | the_value |
| 全局变量 | g_the_value |

配置文件均使用`snake_case`命名
